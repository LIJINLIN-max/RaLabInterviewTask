# Task NestJS Handler


## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.
## Installation

```bash
### You might need --force to install those packages
$ npm install --force
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```

## Deploy
```bash

$ npm run build
$ npm install --force
$ cd dist
$ node main.js

```
