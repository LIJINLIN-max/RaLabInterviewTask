# Task NestJS Handler


## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.
## Installation

```bash
### You might need --force to install those packages
$ npm install --force
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Deploy
```bash
# unit tests
$ npm run build



$ npm install
$ cd dist
$ NODE_ENV=production node main.js

```
