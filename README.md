# Task NestJS Handler


## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.
[Ra Lab] Task Manager
Insert/Find/Delete Task (Update Task)
You may need --force to install the package because I used some old package. But it is doesn't affect the effect.
You may need to build before deploy.
## Installation

```bash
### You might need --force to install those packages
$ npm install --force
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```

## Deploy
```bash

$ npm run build
$ npm install --force
$ cd dist
$ node main.js

```
